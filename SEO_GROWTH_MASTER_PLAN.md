### Page Speed Optimization Checklist
- Inline critical CSS for hero section
- Defer non-critical JS
- Compress/minify CSS and JS
- Use modern image formats (WebP/AVIF)
- Set long cache headers for static assets
- Preconnect to required external origins
- Reduce layout shifts via explicit dimensions
- Lazy-load below-the-fold media

### Core Web Vitals Fixes
- Improve **LCP**: optimize hero text rendering, reduce render-blocking resources
- Improve **INP**: simplify JS event handling and reduce main-thread work
- Improve **CLS**: set width/height on images/buttons and avoid dynamic reflow
